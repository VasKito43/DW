Desenvolvimento web

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar conteúdo na web. Com ela, é possível criar páginas web, inserir textos, imagens, vídeos, links, tabelas, formulários e outros elementos interativos.

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a aparência e o layout de páginas web criadas em HTML. Com ela, é possível definir as cores, fontes, tamanhos, espaçamentos, bordas, efeitos visuais, entre outros aspectos visuais de um site.

JavaScript é uma linguagem de programação utilizada para criar aplicativos web dinâmicos e interativos. Com ela, é possível criar efeitos visuais, validações de formulários, interações com o usuário, manipulação de elementos HTML, entre outros recursos.
